Elastic Calculation Engine

Purpose

The Elastic Calculation Engine combines a number of core features in GigaSpaces, providing some options in how processes are run to show comparisons.

The features are:

  • Master/Worker-based task execution
  • Remote Task Execution
  • Use of the Administration API
  • Elastic Scaling based on memory consumption
  • Deterministic Scaling of Worker tasks
  • Project organization via Maven
